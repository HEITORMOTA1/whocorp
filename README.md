# 🏢 WhoCorp

**WhoCorp** é uma aplicação web desenvolvida para consultar informações de empresas brasileiras por meio do CNPJ de forma rápida, intuitiva e responsiva.

O projeto consome uma API REST para exibir dados atualizados da empresa, oferecendo uma interface moderna e uma experiência simples para o usuário.

---

## ✨ Funcionalidades

* 🔍 Consulta de empresas por CNPJ
* 📄 Exibição de informações cadastrais
* 🏢 Razão Social e Nome Fantasia
* ✅ Situação Cadastral
* 📅 Data de Abertura
* 📍 Endereço Completo
* 🧾 CNAE Principal
* ⚡ Consulta em tempo real via API
* ❌ Tratamento de erros para CNPJs inválidos

---

## 🚀 Tecnologias

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* JSON
* REST API
* Brasil API

---

## 🎯 Objetivo

O WhoCorp foi desenvolvido para aprimorar conhecimentos em desenvolvimento front-end, consumo de APIs REST, manipulação de dados e criação de interfaces modernas voltadas para aplicações do mundo real.
